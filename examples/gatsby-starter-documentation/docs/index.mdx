# gatsby-theme-documentation

A minimalist Gatsby Theme for documentation websites built with
[MDX](https://mdxjs.com) and [Theme UI](https://theme-ui.com).

## Installation

```
yarn add gatsby-theme-documentation
```

## Usage

```js
// gatsby-config.js
module.exports = {
  plugins: [
    'gatsby-theme-documentation'
  ]
}
```
